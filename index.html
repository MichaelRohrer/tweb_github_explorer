Number of commits per contributor


This doughnut graph shows the number of commits per contributors. For example, you can use it to easily find who is the most productive contributor on any GitHub project. Each color represent a different user, the larger it is, the more commits a user has.

Number of commits per day in a week


This bar-chart shows the number of commits per days in a week for a given repository. For example, you can use it to easily find which day is the most productive.


Five most visited repositories


This bar-chart allow you to show which are the five most visited repositories on our site. In order to display this chart, the repo, his owner and the number of times this repo was visited were stored in our database.
